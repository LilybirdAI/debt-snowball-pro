Debt Snowball Pro

Debt Snowball Pro is a modern SwiftUI-based iOS application designed to help users eliminate debt using the proven snowball method.

Overview

Debt Snowball Pro allows users to:
	•	Track multiple debts
	•	Automatically calculate snowball order (smallest balance first)
	•	View total balances and minimum payments
	•	Project payoff timeline and estimated interest
	•	Simulate additional monthly payments (Premium feature)

All calculations are performed locally on-device.

Features
	•	Clean SwiftUI interface
	•	Snowball payoff projection engine
	•	What-If Simulator (Premium via StoreKit 2)
	•	Local data storage
	•	No external payment links
	•	Fully compliant in-app purchase implementation

Tech Stack
	•	SwiftUI
	•	StoreKit 2
	•	Local data persistence
	•	MVVM architecture

Premium

Premium unlocks the interactive What-If Simulator, allowing users to test additional monthly payment scenarios and visualize payoff acceleration.
